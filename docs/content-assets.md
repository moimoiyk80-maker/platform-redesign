# SLI Scientific Content Assets

## 1. Language Rules

### 기본 원칙

* 메뉴, 섹션 제목, 설명, CTA는 국문을 기본으로 사용한다.
* 업계에서 영문 사용이 자연스러운 제품명과 기술 용어는 영문을 유지한다.
* 제품명은 `영문명 + 국문 보조명` 순서로 표기한다.
* 과장된 홍보 표현보다 제품 정보와 사용 목적을 명확하게 전달한다.
* 동일한 행동에는 동일한 문구를 사용한다.

### Product Data Rules

* 제품명은 `영문명`을 기본 제목으로 사용한다.
* 국문명은 제품명 아래 보조 정보로 표기한다.

* 가격은 노출하지 않는다.

### 13.4 Key Features

#### Personnel, Product and Environmental Protection

수직 층류와 전면 유입 기류를 통해 작업자, 시료 및 실험 환경을 동시에 보호합니다.

#### HEPA Filtration System

H14 등급 HEPA 필터를 적용하여 0.3 μm 입자에 대해 99.995% 이상의 집진 효율을 제공합니다.

#### Stable Airflow Control

실시간 풍속 모니터링과 제어 시스템을 통해 작업 공간의 안정적인 기류 상태를 유지합니다.

#### Ergonomic Work Area
